# Система прогнозирования спроса на базе ML-моделей для ИП «Ханафи»

Проект представляет собой прототип системы прогнозирования месячного спроса на товары ИП «Ханафи», реализуемые на маркетплейсах (Wildberries, Ozon). Система построена на базе моделей машинного обучения (CatBoost), использует базу данных SQLite для хранения исторических данных и предоставляет интерфейс для пользователей в виде Telegram-бота.

Основные цели системы:
- снижение частоты и длительности сток-аутов на маркетплейсах;
- улучшение планирования закупок сырья и производства;
- повышение выручки и маржинальности за счёт более точного прогнозирования спроса.

## Технологический стек

- ОС сервера: Ubuntu Server 24.04
- Язык: Python 3.12
- База данных: SQLite 3
- ML-библиотека: CatBoost
- Работа с данными: pandas, numpy, openpyxl
- Telegram: python-telegram-bot (20.x)
- Система контроля версий: Git
